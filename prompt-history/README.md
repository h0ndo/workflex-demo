# Prompt History

Zweck: Nachvollziehbares Gesprächsprotokoll als „Gedächtnis" für das Projekt.

## Regeln

- Dieses Verzeichnis wird vom Agenten fortlaufend aktualisiert.
- Pro Tag gibt es eine Datei im Format `YYYY-MM-DD.md`.
- Einträge sind kurz, klar und in korrigierter Rechtschreibung formuliert.
- Inhalt: Kontext, Entscheidungen, offene Punkte, nächste Schritte.

## Nutzen

- Rückfalloption bei Fehlern oder Missverständnissen
- Schnellere Wiedereinstiege
- Transparente Entscheidungsbasis für Team und Dritte
