# Angular Rules

This folder is intentionally consolidated.

## Canonical files

- `guidelines.md` - binding Angular implementation rules for this project.
- `llms-full.txt` - supplemental reference material (non-binding unless explicitly cited).

## Consolidation note

The former duplicate files (`best-practices.md`, `instructions.md`) were removed to avoid conflicting guidance and drift.

When adding new Angular rules, update `guidelines.md` only.
